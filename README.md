# Mouse Using Hands ✋🖱️  

A Python-based project that uses hand gesture recognition to control mouse movements. This project showcases innovative, touchless computer interaction for accessibility and futuristic user experiences.

## 🚀 Features
- Control mouse movements using hand gestures.
- Real-time gesture recognition with smooth performance.
- Customizable for various gestures and actions.

## 🛠️ Technologies Used
- **Python**: Programming language.
- **OpenCV**: For real-time image processing.
- **MediaPipe**: For efficient hand gesture detection.
- **PyAutoGUI**: To simulate mouse movements.

## 📋 Prerequisites
Before running the project, ensure you have:
1. Python installed on your system.
2. Necessary libraries installed (see Installation section).

## 🔧 Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/RashmikaAkash/Mouse-Using-Hands.git
   cd Mouse-Using-Hands
   ```

2. Install the required Python libraries:
   ```bash
   pip install opencv-python mediapipe pyautogui numpy
   ```

## ▶️ Usage
1. Run the main script:
   ```bash
   python hand_mouse_control.py
   ```

2. Follow on-screen instructions to calibrate hand gestures.

3. Start controlling the mouse using your hands!
